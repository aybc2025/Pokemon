# 🔥 מדריך הקרב של צ'אריזארד

אפליקציית ווב אינטראקטיבית בעברית שמלמדת ילדים בגיל 7 את יסודות קרב הפוקימון TCG עם חפיסת **Mega Charizard X ex**.

## ✨ תכונות

- **לפני הקרב** – שלבים מסודרים להכנת המשחק
- **סדר התור** – הסבר ברור על כל שלב בתור
- **אסטרטגיה** – 5 טיפים חכמים לניצחון
- **חידון אינטראקטיבי** – 5 שאלות לבדיקת הידע
- עיצוב אש מגניב, RTL עברית מלאה, מותאם למובייל

## 🚀 הפעלה מקומית

```bash
# פשוט פתח את הקובץ בדפדפן:
open index.html
```

אין צורך בשרת, בהתקנות, או ב-Node.js. קובץ HTML סטטי מלא!

## 🌐 פרסום ב-GitHub Pages

1. העלה את כל הקבצים ל-repository ב-GitHub
2. לך ל: `Settings → Pages`
3. בחר: `Deploy from branch → main → / (root)`
4. האתר יהיה זמין בכתובת: `https://[username].github.io/[repo-name]`

## 📁 קבצים
